# 背景：人工叡智（Artificial Wisdom / AW）

人工叡智（AW）は、AIの能力を、叡智、自然法則、長期安定性、可逆性、説明責任、生態系との整合性、再生可能性へ接続するための上位フレームワークとして提案されている。

## なぜこのフレームワークが必要なのか

人工知能は、速度、規模、自動化、能力を高める。

しかし、能力が高いことは、叡智があることを意味しない。知的システムが短期的成功、利益、効率、自動化だけを最適化すれば、システムリスクを増幅する可能性がある。

人工叡智は、知的システムに対して「できるか」だけでなく、「すべきか」「どの条件なら許されるか」「長期的に何を壊す可能性があるか」を問う。

## “Artificial Wisdom” という語との関係

“Artificial Wisdom” という英語表現は、広い哲学的・AI的議論の中で既に使われている。

このリポジトリは、その語そのものの世界初使用を主張するものではない。

ここで定義するのは、次の要素を持つ特定の自然法則型人工叡智フレームワークである。

- 六つの理
- 自然法則
- 和ノード思想
- 文明OS概念
- AI調律
- 人工叡智ガードレール・プロトコル

## AIとAWの違い

```text
人工知能は問う：それはできるか？
人工叡智は問う：それはすべきか、どの条件なら許されるか、長期的に何を壊す可能性があるか？
```
